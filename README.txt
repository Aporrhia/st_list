1. Install in XAMPP Control Panel

2. Run Apache and MySQL modules in XAMPP Control Panel

3. Download Student_List .zip folder

4. Extract everything into ...\xampp\htdocs folder

5. In any browser go to http://localhost/phpmyadmin

6. Import 2 databases (web1db, web2db) that are located in \databases folder

7. In any browser go to the http://localhost/web1/public.html
